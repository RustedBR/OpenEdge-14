# Troubleshooting & FAQ

## Frequently Asked Questions

### Gameplay Questions

**Q: How do I get started?**
A: Join a server, select your species and job, then spawn. Check the Getting Started guide for details.

**Q: How do I level up?**
A: Experience comes from:
- Combat (defeating enemies)
- Crafting (successfully creating items)
- Job tasks (completing assigned work)
- Quests (special objectives)
Progress appears in your character panel.

**Q: Can I change my job?**
A: Yes! Visit the job board and apply for a different position. You keep your experience but start a new salary.

**Q: How do I learn spells?**
A: Find spell scrolls and equip them. The spell appears in your action bar. You can also learn from NPCs for a fee.

**Q: What's the best equipment?**
A: Depends on your playstyle:
- Tank: Full Plate + Greatsword + Tower Shield
- DPS: Chainmail + Battleaxe
- Rogue: Leather + Dual Daggers
- Mage: Cloth + Staff + Magical Amulet

**Q: How do I make money fast?**
A:
1. Take a job (8 hours = 4-6 Gold)
2. Hunt monsters (2-4 Gold)
3. Craft items to sell (3-5 Gold)
4. Trade (5-10 Gold per route)

**Q: Can I own a house?**
A: Yes! Save 100+ Gold and visit the Real Estate Office. Buying a house gives passive income!

**Q: Is PvP allowed?**
A: Yes, in wilderness and adventure zones. Towns are safe zones - combat disabled. Check your location!

**Q: How do I join a guild?**
A: Visit the guild hall and apply. Guilds provide benefits and social structure.

### Technical Issues

**Q: The game crashes on startup**
A:
1. Update your graphics drivers
2. Clear game cache: delete `AppData/Local/OpenEdge14/`
3. Reinstall the game
4. Report to support if issues persist

**Q: Lag/stuttering during gameplay**
A:
1. Reduce graphics settings (lower resolution, fewer particles)
2. Close background programs (Discord, Spotify, Chrome)
3. Update drivers
4. Check internet connection (low bandwidth = lag)
5. Verify game files

**Q: Can't connect to server**
A:
1. Check server status (website shows if server is up)
2. Verify firewall isn't blocking the game
3. Try a different server
4. Restart client
5. Check your internet connection

**Q: Inventory full / can't pick up items**
A:
1. Drop some items (right-click → drop)
2. Store items in containers (chests, lockers)
3. Visit bank (store coins to free space)
4. Sell unwanted items
5. Upgrade inventory capacity (requires training)

**Q: Can't cast spells / spells not working**
A:
1. Check mana pool (need enough mana)
2. Verify spell is equipped (appears in action bar)
3. Check spell range (some are melee, some ranged)
4. Ensure target is valid (not behind walls)
5. Wait for spell cooldown

**Q: Equipment disappeared**
A:
1. Check your storage containers
2. Check if items were dropped
3. Did someone steal them? (Check with other players)
4. Were you in a dungeon? (Items reset when leaving)
5. Contact admin if you suspect a bug

### Account & Security

**Q: I forgot my password**
A: Use the "Forgot Password" link on login screen. Check your email for reset instructions.

**Q: My account was hacked**
A:
1. Change your password immediately
2. Contact support with proof of ownership
3. Admin can restore lost items
4. Enable two-factor authentication

**Q: Can I have multiple characters?**
A: Yes! Create additional characters and switch between them. Each has separate progress/inventory.

**Q: Is my data safe?**
A: Yes, encrypted servers and regular backups ensure your data is secure.

**Q: Can I transfer items between characters?**
A: Not directly. Use the trade system or have a trusted friend help transfer.

## Common Problems & Solutions

### Problem: "Entity not found" Error

**Cause**: Item or NPC doesn't exist in this version
**Solution**:
- Restart the game
- Re-download game files
- Try different server

---

### Problem: Can't interact with NPCs

**Cause**: Too far away, NPC is disabled, or bug
**Solution**:
1. Move closer to NPC
2. Try clicking them again
3. Check if NPC is available (some are only active certain hours)
4. Report bug if persists

---

### Problem: Lost a lot of Gold

**Cause**: Died, robbed, or spent carelessly
**Solution**:
1. Check death location (items drop where you die)
2. Report if you suspect theft (admins can help)
3. Use bank storage to prevent future loss
4. Play more safely (avoid dangerous areas solo)

---

### Problem: Quest won't complete

**Cause**: Objective not met correctly, quest bugged, or wrong item turned in
**Solution**:
1. Re-read quest description
2. Verify you have correct item/completed objective
3. Try abandoning and retaking quest
4. Contact quest giver NPC
5. Report bug to admins

---

### Problem: Combat is unfair / Enemies too strong

**Cause**: Wrong level area, poor equipment, or balance issue
**Solution**:
1. Check enemy level (color indicators show threat)
2. Get better equipment
3. Train more (level up)
4. Hunt in easier areas first
5. Team up with others
6. Report balance issues to admins

## Reporting Bugs

### How to Report

**Step 1**: Identify the problem
- What were you doing?
- When did it happen?
- Does it happen every time?
- What's your system?

**Step 2**: Collect information
- Screenshot or video
- Error message (exact text)
- Player name and account
- Timestamp of incident

**Step 3**: Report
- Go to: Help → Report Bug
- Or: Contact support via website
- Or: Message admins in-game

**Step 4**: Follow up
- Check status of report
- Provide additional info if needed
- Test fix when available

### Example Bug Report

```
Title: "NPC Merchant won't accept trades"

Description:
Merchant named Bob at Market won't trade with me.
Every time I try to open his shop, it closes immediately.
This has happened 3 times today.

System: Windows 10, RTX 3060, 16GB RAM
Screenshot: [attached]
Timestamp: 2026-03-23 14:30 UTC

Steps to Reproduce:
1. Go to Market District
2. Find Merchant Bob
3. Right-click to open shop
4. Shop closes immediately
```

## Performance Tips

### FPS Optimization

**Graphics Settings**
- Lower resolution (improves FPS most)
- Disable shadows (helps significantly)
- Reduce particle effects
- Lower view distance
- Disable spell effects

**System Settings**
- Close other programs
- Update graphics drivers
- Verify game isn't running in background
- Restart computer if slow
- Check disk space (need 20GB free)

### Network Optimization

**For High Ping/Latency**
- Close bandwidth-heavy apps (streaming, downloads)
- Use wired connection instead of WiFi
- Check ISP speeds
- Try different server location
- Contact your ISP if consistently slow

**For Disconnects**
- Check internet connection stability
- Restart router
- Use VPN if connection is unstable
- Contact support if issue persists
